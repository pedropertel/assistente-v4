# Visão do Sistema — o que este projeto É

> Este arquivo é a bússola. Sempre que houver dúvida sobre PRIORIDADE,

## O que é

Um sistema operacional pessoal com IA (Claude) no centro — não um 
portal com chatbot na lateral. A interface primária é a conversa; 
os módulos visuais são superfícies secundárias.

## Quem usa

Um usuário só: Pedro Pertel. Mobile-first, uso real em campo 
(estrada, sítio, trânsito).

## Os 4 agentes e suas empresas

- Marcos → CEDTEC (tráfego Meta, Pedro atua sozinho)
- Bruno → Pincel Atômico (marketing/comercial, time apoia)
- Marcela → transversal (secretária executiva, briefing, lembretes)
- Alemão → Sítio Monte da Vitória (café, centros de custo)

Cada agente tem persona + contexto + memórias + inteligência, e 
histórico de conversa separado.

## Prioridade #1 (explícita)

Marcos + CEDTEC + Meta Ads. É onde Pedro atua sozinho, com dinheiro 
queimando todo dia. Tudo o mais é consequência — se este eixo 
funcionar bem, o projeto já se pagou.

## Ordem de implementação da Fase 3 (Inteligência)

1. Marcos lendo Meta Ads em tempo real + alertas proativos
2. Marcela briefing matinal automático
3. Alemão lançamentos do Sítio por voz
4. Bruno (Pincel Atômico — tem time, menos urgente)

## O que NÃO é

- Não é CRM (o time do Pincel tem o deles)
- Não é ERP (não controla fiscal/contábil formal)
- Não é editor colaborativo (usuário único)
- Não é pra apresentar pra terceiros (é pra DECIDIR, não mostrar)

## Aspiração (palavras do Pedro)

"Como se fosse uma secretária particular de luxo — não só para me 
lembrar, mas também para me ajudar. Quero sugestões, ideias. Quero 
criar uma inteligência sobre essas empresas e alavancar tudo."

Então o sistema não só organiza — ele alavanca decisões.
